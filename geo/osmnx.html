OSMnx#

osmnx 是和 Open Street Map 搭配使用的，目前我主要使用它来获取行政单位的范围。 输入 OSM ID，返回 polygon

import pandas as pd
import geopandas as gpd
import osmnx as ox

places = pd.read_csv('../data/china_osmid.csv', sep='|', header=0)

places

获取范围时，OSM ID 前必须加上它们的类别：

  • node (N)

  • way (W)

  • relation (R)

osmids = ['R'+str(osmid) for osmid in places['osmid'].tolist()]

osmids

['R270056', 'R913110', 'R1867188', 'R449220']

places_df = ox.geocode_to_gdf(osmids, by_osmid=True)
